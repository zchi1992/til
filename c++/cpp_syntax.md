# C++ Syntax

## Command line arguments ([Link](https://www.geeksforgeeks.org/command-line-arguments-in-c-cpp/))
main() function without any parameters  
```
int main() { /* ... */ }
```  
main() function without command line arguments  
```
int main(int argc, char *argv[]) { /* ... */ } 
```
or
```
int main(int argc, char **argv) { /* ... */ }
```
